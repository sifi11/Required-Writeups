# How We used AI:
In this competition, we used AI in several ways: most prominently, we used AI to explain concepts to us that we had not encountered yet, as well as to write scripts that we did not have the time or brain capacity to handle at the moment.  We did not use a credit based or paid AI, so we had limited use, specifically, we used `Claude Sonnet 4.6`. The other way we used AI was for help using tools such as Ghidra and LLDB and similarly, we used `Gemini 3 Flash` for help with command formatting. We beleive that AI is a tool, but without us aquiring the knowledge it enlightens us with, it is pointless and useless to use.
